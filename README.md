# Inferno

Endless LoL-targeted skills trainer. Modes: **Teamfight**, Laning Phase, Jungle.

Train the **OODA** loop — Observe, Orient, Decide, Act — under fight pressure without a full client sim.

## Stack

- Vite + TypeScript
- PixiJS (WebGL)
- Fixed-timestep simulation
- Champion icons via Riot Data Dragon

## Run

```bash
cd "Menhir Holdings/Workshop/Inferno"
npm install
npm run dev
```

## Teamfight controls (defaults, remappable)

| Action | Key |
|--------|-----|
| Move / attack | RMB |
| Stop | S |
| Attack-move (no range) | A then LMB |
| Attack-move + range | X then LMB |
| Abilities | Q W E R |
| Actives (Poisson 0–3) | 1 2 3 |
| Ward | 4 |
| Attack champions only | \` |
| Scoreboard | Tab |
